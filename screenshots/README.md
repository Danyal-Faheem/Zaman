# Screenshots Directory

This directory contains screenshots of the Zaman app for documentation purposes.

Please add the following screenshots:

1. **menubar.png** - Menu bar icon with countdown display
2. **popup.png** - Main popup window showing prayer times
3. **settings.png** - Settings view with all options

## Screenshot Guidelines

- Resolution: Retina quality (2x)
- Format: PNG with transparency where applicable
- Size: Keep under 500KB each
- Theme: Show both light and dark mode variants if possible

## Taking Screenshots

1. Run the app
2. Press `Cmd+Shift+4+Space` to capture a window
3. Click on the window to capture
4. Save to this directory with appropriate name
